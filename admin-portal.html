Firebase Configuration

Get this from Firebase Console → Project Settings → Your Apps → SDK setup. Paste the config object. Save and reload to apply.
⚠️ The creator portal must also write applications to Firestore for them to appear here. See setup guide below.

Setup Guide

Step 1 — Firebase Firestore
In creator-portal.html's handleSignup(), add this after validation to write to Firestore:
db.collection('creator_applications').add({
  name: fname+' '+lname,
  email: email,
  university: uni,
  area: area,
  bio: bio,
  status: 'pending',
  appliedAt: firebase.firestore.FieldValue.serverTimestamp()
});
Step 2 — EmailJS
Create a free account at emailjs.com, connect your Gmail/SMTP, create two templates (approval + rejection), and paste the IDs above.

Step 3 — Demo Mode
Without Firestore, this admin portal loads realistic sample applications so you can preview the full UI and decision flow.
